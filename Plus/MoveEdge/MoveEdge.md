# MoveEdge

This is the AHK simpler version of my larger MoveEdge Utility. You only need the exe for use in 32/64 Bit Windows, but the source  (with ico is provided here).

Files 
---
- MoveEdge.exe is a 32bit compiled (version 2.0.19) version for download  
- MoveEdge.ahk is the source file  
- MoveEdge.ico is the embedded resource  
- MoveEdge.png is just an illustration in use  
- MoveEdge - Example.Lnk is just one example of usage (you can use as a template)
