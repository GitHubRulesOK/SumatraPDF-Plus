Theses JS scripts can be run using Recent SumatraPDF or MuPDF MuTool.exe. Based on Strict ES5 they dont have the features one may expect in more graphic JS variations.   
Often it is easier to add a GUI frontend using Windows "Native" C#V5 or HyperText / JScript / VBScript, even CMD hybrids.

# AddLinks.js
A small utility to place text with a Hyperlink on a PDF page.

# AddOverlay.js
A bigger utility to place optionally a box / image and or text as an overlay on one or more PDF pages. Has rotation Alpha graphic states and offsets.
![AddOvarlay.png](https://github.com/GitHubRulesOK/SumatraPDF-Plus/blob/master/Images/AddOverlay.png)

# TextInBox.js
A small annotation workaround to place text as "FreeText" wrapped in a Rectangle (Square) of a different colour. Produced in response to a request see https://github.com/sumatrapdfreader/sumatrapdf/discussions/5645 for usage and options.
